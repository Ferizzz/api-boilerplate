# API boilerplate

API boilerplate made with Node.js, using mongoose.

## Installation

Use the yarn package manager `yarn` to install all required packages.

```bash
cd api-boilerplate
yarn install
```

## Run API

Run API server with one command.


```bash
yarn start
```
or (requires nodemon)
```bash
yarn start-watch
```
